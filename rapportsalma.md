I detta projekt har jag arbetat med utvecklingen av en webbapplikation som fungerar som en publik messageboard. Min roll i gruppen var att arbeta med JavaScript och funktionalitet kopplad till användarinteraktion och datahantering. Jag implementerade funktioner som gör det möjligt för användare att skriva och visa meddelanden, samt kopplade applikationen till Firebase för att spara och hämta data i realtid.
En viktig del av mitt arbete var att säkerställa att formuläret fungerade korrekt, inklusive validering av input så att användare inte kan skicka tomma meddelanden. Jag arbetade även med att förbättra användarupplevelsen genom att lägga till dark mode, vilket gör att användaren kan växla mellan ljust och mörkt tema.
Utöver arbetet i vår egen grupp bidrog jag även till andra gruppers projekt genom att skapa två features. Den första var en like-knapp med räknare, där användare kan gilla meddelanden och se hur många likes de har fått. Den andra var en funktion som visar en “firework”-animation när ett meddelande når ett visst antal likes (varje tionde like). Dessa bidrag gjordes via GitHub genom att forka deras repository, göra ändringar och sedan skicka pull requests.
Genom projektet har jag lärt mig hur man arbetar med Git och GitHub i ett grupparbete, inklusive hur man använder forks, branches och pull requests. Jag har även fått en bättre förståelse för hur man strukturerar kod och samarbetar med andra utvecklare. Projektet har gett mig praktisk erfarenhet av webbutveckling och visat hur viktigt det är med tydlig kommunikation och planering i ett team.
